# Portfolio Website

This repository contains a simple static portfolio website for Lokesh.

## Files

- `index.html` - Main landing page with about, projects, skills, and contact sections.
- `styles.css` - Styling for layout, typography, and responsive design.
- `script.js` - Smooth scrolling and contact form handling.

## Run locally

1. Open `index.html` in your browser.
2. Or use a local server, for example:
   - `python -m http.server 8000`
   - Then visit `http://localhost:8000`

## Customize

Update the content, projects, skills, and contact details to reflect your personal profile.
